AlgoVerse is an interactive **Algorithm and Data Structure Visualizer** built with **React and Vite**.  
It is designed to help learners understand how algorithms work internally through clear, step-by-step visualizations and animations.

---

## Live Demo

https://algo-verse-amber.vercel.app/#/

---

## Features

### Sorting Algorithms
- Bubble Sort  
- Selection Sort  
- Insertion Sort  
- Merge Sort  
- Quick Sort  
- Heap Sort  
- Shell Sort  
- Counting Sort  
- Radix Sort  
- Bucket Sort  

### Searching Algorithms
- Linear Search  
- Binary Search  
- Jump Search  
- Exponential Search  

### Graph Algorithms
- Breadth-First Search (BFS)  
- Depth-First Search (DFS)  
- Dijkstra's Algorithm  
- A* Search  
- Prim's Algorithm (Minimum Spanning Tree)  
- Tarjan's Algorithm (Strongly Connected Components)  

### Tree Algorithms
- Binary Tree Traversals  
    - In-order  
    - Pre-order  
    - Post-order  
- Segment Tree (basic visualization)

### Learning-Focused UI
- Step-by-step animations
- Color legend for states and transitions
- Theory pages for each algorithm
- Adjustable speed and input size
- Clean dark UI for better focus

---

## Tech Stack

- **Frontend:** React 18, Vite  
- **Routing:** React Router (HashRouter)  
- **Styling:** Custom CSS (dark theme)  
- **Deployment:** Vercel  

---

## Run Locally

```bash
git clone https://github.com/PratapOP/AlgoVerse.git
cd AlgoVerse/frontend
npm install
npm run dev
```

Open in browser: `http://localhost:5173`

### Build for Production

```bash
npm run build
npm run preview
```

## Learning Objective

AlgoVerse aims to:

- Make algorithms intuitive instead of memorized
- Help students visualize internal execution steps
- Bridge the gap between theory and implementation
